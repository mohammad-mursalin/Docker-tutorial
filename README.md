# Student App with Docker
This project is a simple Spring Boot application connected to PostgreSQL, demonstrating containerization using Docker and Docker Compose.

## Project Overview
- Spring Boot 3 application
- PostgreSQL database
- Dockerized for easy setup and deployment
- Uses Docker Compose to manage multiple containers


## Prerequisites
- Docker
- Docker Compose
- Java 17+ / Maven
